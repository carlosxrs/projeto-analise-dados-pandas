# Projeto de Análise de Dados com Pandas - Olist

## Sobre o Projeto

Este projeto foi desenvolvido com foco em análise e engenharia de dados utilizando exclusivamente a biblioteca Pandas.

O objetivo foi construir um pipeline de tratamento e análise de dados utilizando dados reais de e-commerce da Olist.

Durante o projeto foram realizadas etapas de:

- Leitura e inspeção dos dados
- Tratamento de valores nulos
- Conversão de tipos de dados
- Criação de métricas de negócio
- Análise de tempo de entrega
- Identificação de pedidos atrasados
- Joins entre tabelas
- Análises por estado, cidade e mês
- Criação de dataset final tratado


## Tecnologias utilizadas

- Python
- Pandas
- VS Code


## Principais Insights

- Mais de 92% dos pedidos foram entregues no prazo
- Agosto foi o mês com maior volume de pedidos
- São Paulo concentrou a maior quantidade de compras
- Alguns estados apresentaram maiores taxas de atraso logístico

## Dataset

Dataset público da Olist disponibilizado no Kaggle.
Link: www.kaggle.com/datasets/olistbr/brazilian-ecommerce
